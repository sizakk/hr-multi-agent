# HR Multi-Agent Analysis System

그룹 HR전략실 멀티에이전트 분석 시스템. 5명의 AI 분석관이 계열사 HR 데이터를 각자의 전문성으로 분석하고, 팀장 에이전트가 회의록 형식의 종합 보고서를 생성합니다.

## 에이전트 구성

| 이름 | 역할 | 담당 데이터 |
|------|------|-------------|
| 김도현 | 인력구조 분석관 | 인력구조, 인원현황 |
| 이서연 | 인력흐름 분석관 | 생산성지표, 인원현황 |
| 박준혁 | 효율성 분석관 | 인당효율 |
| 최예진 | 인재성장 분석관 | 생산성지표, 정규직비율추이 |
| 정민수 | 리스크 분석관 | 전체 5개 파일 교차분석 |
| 한지호 | HR전략실 팀장 | 종합 보고서 작성 |

## 시작하기

```bash
npm install
npm run dev
```

1. Anthropic API 키 입력
2. 5개 CSV 파일 업로드
3. 분석할 회사 선택
4. "분석 회의 시작" 클릭

## 배포

```bash
npm run build
vercel --prod
```

## 입력 데이터

| 파일명 | 내용 |
|--------|------|
| 인원현황.csv | 분기별 정규직/비정규직 현황 |
| 정규직비율추이.csv | 연도별 정규직비율 |
| 생산성지표.csv | S비율, 평균연령, 고령비율, 신규유입률 |
| 인당효율.csv | 인당매출, 인당영업이익, 인건비율 |
| 인력구조.csv | 직급별(S/M/SA/A/JA) 인원 증감 |
